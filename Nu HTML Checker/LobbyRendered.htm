How to Play

Start to play!

  • Press "Create Game" to create a new Game! The private option lets you specify if players need the game code to join or if the game can be accessed from the list.
  • Join a private game with the "Join Game" button and enter the code provided to you.
  • Or choose a game from the list of public games and start the fun!

Create your bingo board!

  • Click on the fields of your bingo board to enter a value.
  • If all fields are filled press on "Submit Board" to finish editing.
  • Watch your opponents by clicking on them in the Player List!

BINGOOOO!

  • If something you wrote happens, click on the field to check it!
  • Once 3 fields in a row, column or diagonal are checked you WIN! Don't forget to shout "BINGOOO"!
  • Watch your opponents by clicking on them in the player List!
